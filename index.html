Github Repo Website

Helping developers get connected with local or remote businesses to contract software engineering help. This is a platform which allows any user to browse posts by businesses looking for any type of software engineering help or a software engineer looking for work. As a business, you are able to create a post outlining what you need (website, database, etc.) including title, description, and rate. Similarly, a logged in engineer is able to post showcasing their skills and rate for businesses to see. On each post, any logged in user is able to make comments intended to help clarify the post and initiate next steps.

Technologies used: Python, Django, Neon PostgressSQL DB, Heroku, JavaScript, HTML5, CSS, Materialize

BLACKJACK

This is the game Blackjack! There is no splitting hands or doubling down. Other than that it is normal blackjack rules. Each player is dealt 2 cards to start, computer/dealer has one face down until their turn. Player will go first and hit (get another card) until they either stay or are over 2. After that, it is the computers turn. The winner is decided by getting blackjack (one jack and one ace) or whoever has the higher number at or under 21.

Technologies used: JavaScript, HTML, CSS, Cards Library
